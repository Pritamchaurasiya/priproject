The subarray must contain at least one element. As a reminder, your code has to be in python """ if not arr: return 0 # Optimization: Use iterator to avoid range() overhead and index lookups iterator = iter(arr) # Safe to call next() because we checked if arr is empty above max_current = max_global = next(iterator) for x in iterator: # Optimization: Explicit conditional check is faster than max() function call current_sum = max_current + x if x > current_sum: max_current = x else: max_current = current_sum if max_current > max_global: max_global = max_current return max_global
